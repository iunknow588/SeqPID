# 统一资金类型判断方法

文档版本：V1.0  
文档状态：统一口径版  
适用范围：连续竞价 + 集合竞价的买入/卖出资金行为判断  

## 1. 文档目标

本文档用于统一三类资金类型的行为定义、判断口径、字段输出和与 PID/P-I-D 解释框架的衔接方式。

本文档替代“买入资金类型分析.md”与“卖出资金类型分析.md”中的分散口径，形成单一有效标准。

## 2. 三类资金的统一定义

本系统不按真实账户身份定义资金类型，而按订单行为对价格走势的作用方式定义。

### 2.1 游资

定义：

> 试图快速影响股价走势的资金，定义为游资。

行为内涵：

1. 强调速度
2. 强调主动性
3. 强调对价格的快速推动、快速压制、快速抢筹、快速出货
4. 愿意为立刻改变价格方向支付更高冲击成本

典型市场表现：

1. 大额主动吃单
2. 多档穿越买入或卖出
3. 短时间内显著改变成交价或盘口重心
4. 集合竞价中以大额委托快速抬高或压低虚拟成交价

### 2.2 量化

定义：

> 试图缓慢改变价格走势的资金，定义为量化。

行为内涵：

1. 强调节奏控制
2. 强调分批执行
3. 强调低冲击、持续性、可重复
4. 倾向于通过多笔、连续、相对克制的交易逐步改变价格重心

典型市场表现：

1. 中大额但不激进的成交
2. 小幅跨价或贴价成交
3. 多窗口持续净买入或净卖出
4. 盘口回补、节奏稳定、冲击后价格变化较平滑

说明：

本文档不再使用“量化/机构”混合表述，统一写作“量化”。

### 2.3 散户

定义：

> 试图适应既有价格变化的资金，定义为散户。

行为内涵：

1. 强调跟随
2. 强调被动适应
3. 常在既有涨跌趋势下追涨、杀跌、补仓、止损
4. 对价格的影响更多体现为顺应已发生走势，而不是主动塑造走势

典型市场表现：

1. 小额零散成交
2. 在价格已经明显变化后才跟进
3. 急迫小单追价买入或恐慌小单砸盘卖出
4. 集合竞价中小额跟风挂单

## 3. 总体判断原则

### 3.1 先看“是否主动塑造走势”，再看“如何塑造”

统一判断优先级：

1. 是否试图快速影响价格：若是，优先判游资
2. 是否试图缓慢改变价格：若是，优先判量化
3. 是否主要在适应现有走势：若是，优先判散户

### 3.2 先看行为强度，再看金额规模

判断资金类型时，行为作用强于绝对金额。

即：

1. 大额但不主动、不急迫、不跨价，不应直接判为游资
2. 小额但极端追价、极端杀跌，只能作为散户情绪种子，不应直接升级为游资
3. 中大额、分批、连续、低冲击，更应优先解释为量化

### 3.3 金额只是辅助，不是真值

金额规模只用于增强置信度，不单独决定资金类型。

### 3.4 输出的是“行为锚点”

规则层输出的是：

1. 游资强锚点
2. 量化锚点或量化候选流
3. 散户种子流
4. 量化+散户混合池

它们用于后续 PID 反解与贡献判断，不等价于真实账户真值。

## 4. 判断维度

统一使用以下行为维度：

| 维度 | 含义 | 主要用途 |
| --- | --- | --- |
| 主动性 | 是否主动吃单/砸盘 | 区分主动塑造与被动适应 |
| 价格冲击强度 | 是否跨价、跨几档、虚拟成交价偏离多大 | 区分游资与非游资 |
| 执行节奏 | 是瞬时集中还是分批持续 | 区分游资与量化 |
| 金额规模 | 单笔或阶段累计金额大小 | 增强游资/量化/散户置信度 |
| 趋势相对位置 | 是领涨、领跌还是跟随 | 区分主动塑造与顺势适应 |
| 盘口后效应 | 冲击后是否快速回补、是否持续施压 | 区分量化与散户 |
| 时间约束 | 是否处于不可撤单集合竞价等高约束场景 | 调整证据权重 |

## 5. 连续竞价判断方法

## 5.1 买入行为判断

连续竞价买入先定义：

1. `P_order`：委托价
2. `P_best_ask`：委托时卖一价
3. `tick_size`：最小变动价位
4. `N_tick_buy = floor((P_order - P_best_ask) / tick_size)`

其中：

1. `N_tick_buy >= 1` 表示主动向上跨价
2. `N_tick_buy >= 3` 表示强烈快速推动价格

买入判断原则：

### A. 判游资买入

满足以下特征越多，越倾向判为游资买入：

1. 主动买入
2. `N_tick_buy >= 3`
3. 单笔或短时累计金额较大
4. 成交后价格立即上移
5. 明显试图快速抬升价格

统一解释：

> 快速改变价格走势的买入行为，判为游资买入。

输出：

- `buy_ch_anchor += signed_amount`

### B. 判量化买入

满足以下特征越多，越倾向判为量化买入：

1. 主动或半主动买入
2. `1 <= N_tick_buy < 3` 或贴价持续成交
3. 金额中大
4. 多笔分散、节奏平稳
5. 价格被缓慢抬升，而不是瞬间冲高

统一解释：

> 缓慢改变价格走势的买入行为，判为量化买入。

输出：

- `buy_q_anchor += signed_amount`
或进入
- `U_q`

### C. 判散户买入

满足以下特征越多，越倾向判为散户买入：

1. 小额成交
2. 发生在价格已经上涨之后
3. 追价但金额小
4. 行为零散、持续性弱
5. 更像跟随上涨，而不是发动上涨

统一解释：

> 适应既有上涨走势的买入行为，判为散户买入。

输出：

- `buy_retail_seed += signed_amount`
或进入
- `U_retail`

## 5.2 卖出行为判断

连续竞价卖出先定义：

1. `P_order`：委托价
2. `P_best_bid`：委托时买一价
3. `N_tick_sell = floor((P_best_bid - P_order) / tick_size)`

其中：

1. `N_tick_sell >= 1` 表示主动向下砸盘
2. `N_tick_sell >= 3` 表示强烈快速压低价格

卖出判断原则：

### A. 判游资卖出

满足以下特征越多，越倾向判为游资卖出：

1. 主动卖出
2. `N_tick_sell >= 3`
3. 单笔或短时累计金额较大
4. 成交后价格立即下移
5. 明显试图快速打压价格、出货或砸盘

统一解释：

> 快速改变价格走势的卖出行为，判为游资卖出。

输出：

- `sell_ch_anchor += signed_amount`

其中卖出方向应记为负值。

### B. 判量化卖出

满足以下特征越多，越倾向判为量化卖出：

1. 主动或半主动卖出
2. `1 <= N_tick_sell < 3` 或贴价持续成交
3. 金额中大
4. 多笔分散、节奏稳定
5. 价格被缓慢压低，而不是瞬间砸穿

统一解释：

> 缓慢改变价格走势的卖出行为，判为量化卖出。

输出：

- `sell_q_anchor += signed_amount`
或进入
- `U_q`

### C. 判散户卖出

满足以下特征越多，越倾向判为散户卖出：

1. 小额成交
2. 发生在价格已经明显走弱之后
3. 急迫砸盘但金额小
4. 行为零散、持续性弱
5. 更像顺着下跌去止损或恐慌卖出

统一解释：

> 适应既有下跌走势的卖出行为，判为散户卖出。

输出：

- `sell_retail_seed += signed_amount`
或进入
- `U_retail`

其中卖出方向应记为负值。

## 6. 连续竞价分类矩阵

| 行为特征 | 主要解释 | 资金类型 | 输出去向 |
| --- | --- | --- | --- |
| 大额 + 主动 + 强跨价 + 立即改变价格 | 快速塑造走势 | 游资 | `CH_anchor` |
| 中大额 + 持续 + 小跨价/贴价 + 缓慢推动价格 | 缓慢改变走势 | 量化 | `Q_anchor` / `U_q` |
| 小额 + 跟随已有上涨/下跌 + 持续性弱 | 适应既有走势 | 散户 | `Retail_seed` / `U_retail` |
| 非游资且难以清晰分出量化/散户 | 暂不强分 | 量化+散户混合池 | `M_qr` / `U_mix` |

## 7. 集合竞价判断方法

集合竞价不能直接用连续竞价的“跨几档”定义，因此改用“虚拟成交价偏离 + 委托行为强度 + 是否不可撤单”判断。

## 7.1 统一原则

集合竞价中优先看：

1. 是否处于不可撤单阶段
2. 是否显著推动虚拟成交价变化
3. 推动方式是快速、集中，还是平稳、持续
4. 是主动塑造开盘/收盘位置，还是跟随已有方向

## 7.2 关键变量

定义：

1. `P_virtual`：虚拟成交价
2. `P_prev_close`：前收盘价
3. `delta_virtual = (P_virtual - P_prev_close) / P_prev_close`
4. `V_stage`：集合竞价阶段累计委托金额
5. `V_net_locked`：不可撤单阶段净新增委托金额

其中：

1. `V_stage` 反映总规模
2. `V_net_locked` 反映真实约束下的有效意图

## 7.3 集合竞价买入

### 判游资买入

满足以下特征越多，越倾向判为游资集合竞价买入：

1. 不可撤单阶段大额净新增买单
2. 快速抬高虚拟成交价
3. 对开盘/收盘定价位置有明显塑造意图
4. 价格偏离大且集中发生

解释：

> 在集合竞价中试图快速改变定价位置的买入行为，判为游资买入。

### 判量化买入

满足以下特征越多，越倾向判为量化集合竞价买入：

1. 不可撤单阶段有稳定净买入
2. 虚拟成交价偏离中等
3. 金额中大
4. 更像平稳建仓，而不是强势抢筹

解释：

> 在集合竞价中试图缓慢改变定价位置的买入行为，判为量化买入。

### 判散户买入

满足以下特征越多，越倾向判为散户集合竞价买入：

1. 小额挂单
2. 跟随高开预期
3. 缺乏主导定价能力
4. 更像适应即将形成的开盘价格

解释：

> 在集合竞价中适应既有高开或转强预期的买入行为，判为散户买入。

## 7.4 集合竞价卖出

### 判游资卖出

满足以下特征越多，越倾向判为游资集合竞价卖出：

1. 不可撤单阶段大额净新增卖单
2. 快速压低虚拟成交价
3. 对开盘/收盘定价位置有明显打压意图
4. 价格偏离大且集中发生

解释：

> 在集合竞价中试图快速改变定价位置的卖出行为，判为游资卖出。

### 判量化卖出

满足以下特征越多，越倾向判为量化集合竞价卖出：

1. 不可撤单阶段稳定净卖出
2. 虚拟成交价偏离中等
3. 金额中大
4. 更像平稳减仓，而不是瞬时砸盘

解释：

> 在集合竞价中试图缓慢改变定价位置的卖出行为，判为量化卖出。

### 判散户卖出

满足以下特征越多，越倾向判为散户集合竞价卖出：

1. 小额挂单
2. 跟随低开预期或利空预期
3. 缺乏主导定价能力
4. 更像适应即将形成的弱势价格

解释：

> 在集合竞价中适应既有低开或转弱预期的卖出行为，判为散户卖出。

## 8. “试盘/虚假挂单”处理原则

可撤单集合竞价阶段的大额挂单，不直接纳入净额主口径。

若出现：

1. 可撤单阶段大额挂单显著推动虚拟成交价
2. 临近不可撤单前快速撤单
3. 不可撤单阶段未保留对应净意图

则应解释为：

- `试盘`
- `虚假挂单`
- `异常信号`

处理方式：

1. 记为诊断标签
2. 不计入主净额
3. 仅作为低权重辅助证据

## 9. 特殊情形修正

## 9.1 接近涨跌停

在接近涨跌停时，档位深度不再能完整反映真实攻击性。

因此应改为：

1. 观察剩余价格空间占比
2. 观察在极小剩余空间内是否仍出现大额主动单
3. 对规则结果降权或重标化

修正原则：

1. 接近涨停的大额主动买入：可提升游资权重
2. 接近跌停的大额主动卖出：可提升游资权重
3. 接近涨停的小额追价买入：更偏散户
4. 接近跌停的小额恐慌卖出：更偏散户

## 9.2 方向恢复不可靠

若主动买卖方向无法稳定恢复，则：

1. 不做强资金类型判定
2. 只进入混合池规模统计
3. 置信度下调

## 9.3 单笔极端但后续无持续

若仅有单笔极端冲击，但后续无持续成交、无盘口跟随、无价格后效应，则：

1. 可视为弱游资候选
2. 不直接作为高置信游资强锚点

## 10. 统一字段规范

## 10.1 原子规则输出

建议输出以下字段：

| 字段 | 含义 |
| --- | --- |
| `buy_ch_anchor` | 买入方向游资强锚点金额 |
| `sell_ch_anchor` | 卖出方向游资强锚点金额 |
| `buy_q_anchor` | 买入方向量化锚点金额 |
| `sell_q_anchor` | 卖出方向量化锚点金额 |
| `buy_retail_seed` | 买入方向散户种子金额 |
| `sell_retail_seed` | 卖出方向散户种子金额 |
| `mix_qr_signed` | 无法强分时进入量化+散户混合池的净额 |
| `buy_aggressiveness` | 买入行为主动影响价格的强度分 |
| `sell_aggressiveness` | 卖出行为主动影响价格的强度分 |
| `direction_reliability` | 主动方向恢复可靠性 |
| `auction_locked_net` | 不可撤单阶段净新增委托金额 |

## 10.2 统一净流口径

合并后进入 PID 的建议变量：

\[
U_{ch} = buy\_ch\_anchor + sell\_ch\_anchor
\]

\[
U_q = buy\_q\_anchor + sell\_q\_anchor
\]

\[
U_{retail} = buy\_retail\_seed + sell\_retail\_seed
\]

\[
\widehat{M}_{qr} = mix\_qr\_signed
\]

其中卖出金额使用负号。

## 11. 与 PID 主设计的衔接关系

本规则层与主算法设计的关系如下：

1. 本规则层负责生成行为锚点和种子流
2. `U_ch` 主要承接高置信“快速改变走势”的资金
3. `U_q` 主要承接高置信“缓慢改变走势”的资金
4. `U_retail` 主要承接高置信“适应走势”的资金
5. 其余不够清晰的非游资流量先进入 `M_qr`
6. 最终三类贡献仍由 PID 联盟方程和状态空间模型共同解释

也就是说：

> 规则层负责给出高置信行为起点，模型层负责给出结构闭合与最终贡献解释。

## 12. 最终判定口径

统一总结如下：

1. 试图快速影响股价走势的资金，判为游资
2. 试图缓慢改变价格走势的资金，判为量化
3. 试图适应既有估计变化的资金，判为散户

对应到可观测行为：

1. 快、猛、主动、强冲击：游资
2. 稳、缓、连续、低冲击：量化
3. 跟、随、零散、适应性：散户

该口径适用于买入与卖出两侧，区别只在于方向符号不同：

1. 买入为正贡献
2. 卖出为负贡献
