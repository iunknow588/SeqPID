# 前端说明

这是 TongStock 的 Web UI 目录，负责把行情、K 线、分时、盘口、信号解释做成可视化页面。

## 页面职责

- 行情总览
- 个股详情
- 信号筛选
- 指标分析

## 对接目标

后续建议对接你项目里的行情聚合层，让前端直接消费统一 JSON：

- 快照
- 分时
- 五档盘口
- 分笔成交
- 模式识别结果
- 资金类型判断

## 开发方式

```bash
npm install
npm run dev
```

如果后端不在同域，设置 `VITE_API_BASE` 指向 API 地址。

## 备注

这里保留图表、表格、筛选和详情页能力，后续可替换为你项目自己的数据源与业务接口。
